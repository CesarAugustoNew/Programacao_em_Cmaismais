Praticar os conceitos aprendidos em aula
Servir como material de consulta futura
Demonstrar habilidades em C++
Compartilhar conhecimento com outros estudantes
🖥️ Como Executar os Projetos

Para rodar os códigos, você pode utilizar qualquer compilador C++, como:

🔹 Code::Blocks
🔹 Dev-C++
🔹 Visual Studio Code (com extensão C++)

Passos básicos:
# Clone o repositório
git clone https://github.com/CesarAugustoNew/Programacao_em_Cmaismais.git

# Acesse a pasta
cd Programacao_em_Cmaismais

# Compile um arquivo (exemplo)
g++ nome_do_arquivo.cpp -o programa

# Execute
./programa
